#ApiClient Command Line
## **UPDATES COMING SOON**
##Summary
This is a lightweight API client that handles GET requests. Api Client has a command line interface that prompts users to provide an endpoint for the API that they are trying to connect to. If the request is successful, the user will be prompted with a menu, asking if they would like the data to be pretty printed, and/or to save the data to a database.
